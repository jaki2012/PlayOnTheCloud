# Angular2Webpack
A front-end framwork built with Angular2 and Webpack!

### Usage: 
Before you do anything, execute the command below:
```
  npm install
```

Then:
* development mode
```
  npm start
```
* production mode 
```
  npm run build
  cd dist 
  http-server (you have to do npm install http-server -g beforehand)
```

* test mode
